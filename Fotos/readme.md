# Aplicación Web de Fotografía

## Descripción

Desarrollo de una aplicación web simple para presentación de fotografía con estructura modular basada en Django.

## Estructura del Proyecto

El proyecto consta de dos páginas principales:

1. **Página Principal (Raíz)**
   - Ubicación: `/`
   - Contenido: Logotipo e información del sitio
   - Función: Landing page

2. **Galería de Portfolio**
   - Ubicación: `/portfolio` o ruta secundaria
   - Contenido: Serie de imágenes en formato portfolio
   - Función: Exposición de trabajos

## Especificaciones Técnicas

- Framework: Django
- Versión: Web 1.0
- Características: Páginas estáticas con contenido visual
